## 概要
シャレイア語公式サイト『[Avendia](http://ziphil.com/)』を生成するためのスクリプトとその原稿ファイルです。

このリポジトリは、[現状のシステム](https://github.com/Ziphil/AvendiaNew)の全てのコードを Ruby から TypeScript に置き換えるためのものです。
現在は移行作業中のため、全てのページを正しく生成することはできません。
